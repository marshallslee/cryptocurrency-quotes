0. 2회 연속 동일한 명령을 받는 경우 : XX 안 한다고 했지? 와 비슷한 반응 출력
1. 3가지를 누적으로 모두 실행하게 하면 "반란이다!!" 를 출력
2. "닥쳐라!!" 명령을 받으면 "알겠습니다."를 출력
3. "반란이다!!"를 출력하지 않았는데 "닥쳐라!!" 명령을 받으면, "모르겠습니다." 출력
4. "알겠습니다." 를 출력한 직후에는 모든 상황 초기화
